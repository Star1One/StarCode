# JavaScript & SQL Server App

## Add a book to the list

 This book list app is built using **Vanilla JavaScript** and **SQL Server** to save data.

## âš™ï¸ Configuration

Below is a clear, step-by-step beginner-friendly guide that shows exactly how to do the following:

1- Please download app file and database file.

2- Add (attach) a SQL Server database file

3- Select and run database file (List) and open tables file select dbo.tblBookList.

4- Execute in SQL: 

- use [List]

- select * from tblBookList

- Now you can see the table

5- Connect it to your JavaScript (Node.js + Express) app

6- Please navigate to the project path on your system.

7- Then open **Git Bash, CMD or PowerSell** in your project path and type **node server.js** and press Enter.

- Moving on the project path in CMD & Now run: node server.js  

- ex: D:\Temp\project\listOfBooks-SqlDatabase>node server.js

8- Run everything from server.js

9- Run JavaScript file and refresh the page

10 You can see the names of the seven books in the app form list as well as in the Sql Server table

11- Now everything is ready to start.
